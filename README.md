# Исследовательский анализ рынка заведений общественного питания Москвы

## Описание проекта
Проект посвящён исследовательскому анализу рынка заведений общественного питания в Москве.
Цель — на основе открытых данных проанализировать структуру рынка, выявить ключевые
характеристики заведений и определить факторы, связанные с рейтингом и ценовым сегментом.

## Используемые инструменты
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Этапы анализа
- Предобработка и очистка данных
- Исследовательский анализ данных (EDA)
- Анализ распределений и взаимосвязей
- Визуализация ключевых показателей
- Формирование выводов и рекомендаций

## Основные результаты
- Проанализирована структура рынка заведений общественного питания Москвы
- Выявлены различия между сетевыми и несетевыми заведениями
- Исследована зависимость рейтинга от вместимости и ценового сегмента
- Определены факторы, наиболее связанные с рейтингом заведений
